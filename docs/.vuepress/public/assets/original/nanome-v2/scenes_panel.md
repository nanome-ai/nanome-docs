# Scenes Panel

The Scenes Panel allows you to manage scenes within a workspace. Scenes store specific views and representations of your structures, making it easy to switch between different perspectives during a session.

<vimg src="nanome-v2/scenes_panel_overview.jpg" />

## Controls

The top section of the Scenes Panel includes the following controls:

- **Workspace Name**: Displayed at the top of the panel, along with a lock icon indicating permission status.
- **Create Scene (+)**: Add a new scene to the workspace.
- **Duplicate Scene**: Copy the current scene, including all its representations and settings.
- **Set Scene POV**: Save your current perspective as the point of view for this scene.
- **Enable/Disable Scene POV**: Toggle whether switching scenes automatically adjusts your viewpoint.
- **Previous / Next Scene**: Navigate between scenes using the arrow buttons.
- **Scene Options (â€¦)**: Access rename and delete options for individual scenes.
- **Reorder Scenes (=)**: Drag to rearrange scenes in the list.

## Users Section

At the bottom of the Scenes Panel, the **Users** section shows who is currently in the workspace. Each user is assigned a personal color, displayed on their name tag and their crown icon. A **Share Workspace** button allows you to invite others, and a **Compact Panel** toggle (>>) collapses the panel for more viewing space.

## Multi-User Scene Indicators

<vimg src="nanome-v2/scenes_panel_multiuser.jpg" />

When multiple users are in a workspace, colored dots on each scene indicate which users are currently viewing that scene. Each user's personal color corresponds to their name tag in the Users section. In the example above, both Joe (yellow) and Steve (blue) are viewing Scene 1 ("Helices"), with each user seeing their own color as the current-user indicator.

## Spotlight Mode

<vimg src="nanome-v2/scenes_panel_spotlight.jpg" />

Clicking **Spotlight Me** activates Spotlight Mode, which makes your view the shared perspective for all followers in the session.

- **Spotlighter's View (Steve)**: The "Spotlighting 1/1" indicator shows Spotlight Mode is active and how many users are following. Users following are listed with an eye icon next to their name.
- **Follower's View (Joe)**: When following a spotlighter, the follower sees the spotlighter's name with a crown icon, along with a **Reset Orientation** button and a **Stop** button to exit follow mode.

This system allows seamless collaboration without a designated presenter — users can independently spotlight, follow, or free-look at any time.
