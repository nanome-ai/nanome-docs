# MD Trajectory Playback

### Video: MD Trajectory Playback

<YoutubeEmbed videoId="cNW8DnFap6A" title="Nanome v2.5 Quick Tips: MD Trajectory Playback" />

Stream molecular dynamics trajectories from the web to standalone XR headsets in real time.

Because the trajectory is streamed from Nanome's web platform, the headset needs no local files and no high-end GPU — the tutorial shows a live trajectory playing on a Quest 3.

Multiple users on different devices — Quest 3, Vision Pro, phones, and desktops — can watch the same simulation simultaneously, making MD playback a collaborative, cross-platform experience.
