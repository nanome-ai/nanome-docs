# Loading Structures

### Video: Loading Structures from the Web

<YoutubeEmbed videoId="BkZ6R8fPbgQ" title="Nanome v2.5 Quick Tips: Loading Structures from the Web" />

Start a new workspace on app.nanome.ai and load your first protein in under three minutes.

The tutorial covers:

- **Create a project and workspace** — Set up somewhere to load your structures into.
- **Import from the RCSB** — Load structures directly by PDB ID.
- **Upload from your computer** — Drag and drop files into the workspace.
- **Import from chat** — Pull in molecules from previous MARA chats.

Nanome automatically generates a pocket view for any ligand-containing structure you load.

The web app accepts the full range of supported file types — structures, session files, trajectories, and electrostatic maps. For the complete list, see [Supported File Formats](/nanome_web/fileformats).
